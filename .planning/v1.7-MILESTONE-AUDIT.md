## 3. Integration & System Integrity

### Verified Wiring (via Integration Subagent)
- **Legal Engine**: `NomineeService` correctly consumes `LegalParamSet`. All calculation constants (OT multipliers, CCSS rates, workday hours) are now dynamic.
- **Snapshot System**: Approval of any payroll now triggers an atomic capture of all active parameters into `vpg_payroll_param_snapshots`.
- **Shift Logic**: Shift caps (6h for Nocturna, 7h for Mixta) are dynamically resolved per employee and correctly applied by the motor.
- **Aguinaldo Projection**: Accrual logic (Sum/12) is centralized in `AguinaldoService` and surfaced consistently in the Wizard and Profile.
- **Auditable Configuration**: Changes to critical parameters require password verification and trigger dashboard notifications for the team.
- **Modular UI**: The Employee Profile has been successfully refactored into a tab-based system with real-time data fetching and actions (PDF downloads, event management).
